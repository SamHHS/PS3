Kwartje

Case description

The kwartje case has been a assignment for school as well, it was used to create a website where a brewery is able to show of their products and invite the visitors to become customers. This was also a solo project so everything that has been done such as designing the website and developing the website has all been part of a solo project.

Sketches

The first part of the process is sketching out possible idea's for the website. This can be done on paper or digitaly, in this case its been done on paper at first and transitioned into digital later on in the process.

Wireframe

Wireframes are an important part of the process, this is the first time youll actually see the shapes and layout of the website you are designing. This leaves enough time and space for improvement which is a big part of this step.

HiFi prototype

The HiFi prototype is a great tool when designing a website or application. The HiFi prototype will give a good insight in the style of the website and the overall look at the end. Therefore it plays a major part when desining a website.
